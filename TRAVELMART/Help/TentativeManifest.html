               2.1.1 Under Homepage, Point the mouse cursor to Hotel View Tab.



                       2.1.1.1 Select Tentative Manifest.



                       2.1.1.2 Click on the Hotel Branch drop down list and select specific hotel branch name.

                           * Tick View Advance Search to view filter options.

                               

                       

                       2.1.1.3 Display the list of Hotel Manifest for the Selected branch.

                       2.1.1.4 To view the Crew Information, Select and Click the name of crew link.

                       2.1.1.5 Click the Lock button to lock manifest. Lock means that the list has been finalized for the                                         particular Manifest Type by the Hotel Specialist. You can only lock a manifest                                                           for a particular type once.

                       2.1.1.6 Click the Export Tentative Manifest button to view or Save records in Excel file.
