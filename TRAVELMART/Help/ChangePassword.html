TravelMart Website ›› Homepage ››




               2.2.1 Under Homepage, Click the change password link to view change password window.



2.2.1.1 Display Change Password page.

2.2.1.2 Enter your current password on the Password field and your new password on                  the New Password and Confirm New Password fields. New Password                                and Confirm New Password fields must match.

2.2.1.3 Click the Save button to change your password.

2.2.1.4 Click the Cancel button to disregard changes and return to previous page.
