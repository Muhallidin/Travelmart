               2.1.1 Under Homepage, Point the mouse cursor to Hotel View Tab.




                       2.1.1.1 Select Locked Manifest.



                       2.1.1.2 Click on the Hotel Branch drop down list, then Select specific hotel branch name.

                       2.1.1.3 Click on the Manifest Type drop down list, then Select Manifest Type.




                       2.1.1.4 Click on the second Manifest Type drop down list, then Select Manifest Type to enable                                           Compare button. Click Compare button to compare selected manifests.

                       2.1.1.5 Click the View button to display records under Locked Manifest.

                       2.1.1.6 To view the Crew Information, Select and Click the name of crew link.




                       2.1.1.7 Click the Export Locked Manifest button to view or Save records in Excel file.

                       2.1.1.8 Click the View E-mail Address to open Hotel Manifest E-mail Window and view                                                       TO and CC e-mail address(es).  

                                                               

                                              

                               2.1.1.8.1 Add or remove e-mail addresses as needed. At least one e-mail address is                                                              required.

                               2.1.1.8.2 Add or remove e-mail addresses to be copied on the e-mail as needed. This field is                                                optional.

                               2.1.1.8.3 Click Send button to send Locked Manifest to designated Recipient(s) and/or                                                          Carbon Copy list.

                               2.1.1.8.4 Click Save & Send button to save recipients and/or carbon copy list and send                                                          Locked Manifest. Saved list of e-mail address(es) will be used as default intended                                                  recipients on the next e-mail.        

                                         

                       2.1.1.9 Tick the Send as .pdf checkbox to send Locked Manifest in .pdf file format instead of the                                       default excel (.xls).

                       2.1.1.10 Click the Send E-mail button to send Locked Manifest to default recipient(s)                                                             as viewed on the Hotel Manifest E-Mail (2.1.1.8).
